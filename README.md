# Far-East-NEST | Tactical Support Module
**極東ネスト 居住民向け：素材逆引き・クラフト支援アプリケーション**

---

## 🛰 概要
本モジュールは、極東ネスト（Far-East-NEST）のドリフターおよび居住民に対し、地上探索で得られた物資の有効活用を支援するために配布されている公式サポートアプリです。

「持ち帰った素材が何に使えるのか」を即座に照合し、複雑なクラフト工程における最終的な必要リソース（末端素材）を逆引き算出します。

## ⚙️ 主な機能
* **素材逆引き（Reverse Lookup）**: 素材名から、それを使用する全てのクラフト項目を抽出。
* **再帰構成解析（Recursive Analysis）**: 中間素材を末端コンポーネントまで分解し、合計数を提示。
* **高効率UI**: 視認性を重視したフラットデザイン。

## 📑 データ同期プロトコル
本アプリのデータベースは、有志ドリフターによる観測データ（Wiki）を基底プロトコルとして、以下の2段階のプロセスで構築されています。

### 1. アイテムマスタ・プロセッサ (`convItem.py`)
名称、レアリティ、入手場所、価格などの「基本辞書」データを生成します。
* **入力**: `itemALL.html`
* **出力**: `item_master.csv`

### 2. レシピ相関・プロセッサ (`conv.py`)
素材同士のつながり（どの素材から何ができるか）を定義する相関データを生成します。
* **入力**: `itemALL.html`
* **出力**: `craft_data.csv`

## 🛠 データベース更新手順（管理者用）
最新の観測データを反映させる際は、以下のスクリプトを実行してください。
※パス設定は各自のローカル環境（`scripts`フォルダ内等）に合わせて適宜調整すること。

```powershell
# アイテム基本情報の更新
python scripts/convItem.py

# クラフト相関データの更新
python scripts/conv.py
```
